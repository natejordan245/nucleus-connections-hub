# Nucleus Connections Hub

AI-powered talent ↔ startup matching for the Utah innovation ecosystem.

Hackathon entry for the Nucleus Utah bounty: build a platform that matches operators, executives, students, and advisors to Utah deep-tech startups and university spinouts — with explainable matches and Utah-specific ecosystem signal that generic job boards can't see.

## Approach

- **Embeddings** over rich profile text for semantic candidate retrieval
- **LLM re-ranker + explainer** ("Why was I matched?") for transparency
- **Structured gates** (stage, availability, equity vs cash) as hard filters
- **Utah ecosystem graph** — U of U / BYU / USU spinouts, Silicon Slopes alumni, shared advisors, university lineage
- **Squarespace + Affinity** integration for drop-in deployment into Nucleus's existing stack
