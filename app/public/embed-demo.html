Embed demo

This page shows what a Squarespace / Webflow / plain-HTML site sees when they embed the Connections Hub signup widget. Paste the snippet below into a code block and you're done — the button opens a modal that walks the visitor through role → profile → account, then drops them into Connections Hub already signed in.

Paste this snippet

<script async src="https://app-two-swart-3c3ey9rx1h.vercel.app/embed.js"></script>
<button data-nucleus-embed>Join Connections Hub</button>

Drop this into a Squarespace code block, a Webflow embed element, or a plain HTML page. The script tag finds every data-nucleus-embed button on the page and wires it up.
